Desafios

“Se optar pelo prazer do crescimento, prepare-se para sofrer.” _Friedrich Nietzsche

Cada etapa cumprida durante as aulas, tornou desafiador o processo em me tornar uma Desenvolvedora Web. As aulas ministradas pela professora, Gil Chaves, sob mentoria da, Vanessa Mainara, foram fundamentais para adquirir conhecimento na linguagem de programação e alguns cabelos brancos… foi cada desafio proposto, que chegar aqui hoje, e apresentar esse projeto final, me deixou quase careca… Brincadeiras a parte, foi uma honra e um privilégio cumprir cada etapa do processo e apresentar os desafios propostos em cada aula e saber que fui bem orientada sobre as instruções e o incentivo dessas duas mulheres, que hoje se tornaram exemplo profissional e referência para mim.

INICIANDO HTML

#HTML #CSS

Ver Repositório

Primeiro desafio proposto em aula. Aqui pude aprender os fundamentos de HTML e desenvolver o meu primeiro layout utilizando as tags semânticas da linguagem. Neste desafio, o projeto era criar um arquivo HTML com as estruturas básicas e vincular um arquivo CSS para dar identidade ao layout da página.

#PARTIU HTML

#HTML #CSS

Ver Repositório

Criei um projeto de um site de Festival de Música Eletrônica fictícia, o Desert Moon Festival, abordando as tags semânticas em HTML e um layout que tenha: header, nav, main, section, article, footer, div. Todo o visual foi formatado no CSS utilizando os conceitos e fazendo o vínculo com o HTML dando vida ao projeto.

FLEXBOX

#HTML #CSS

Ver Repositório

Utilizando o flexbox, criei uma estrutura básica em HTML e estilizei o conteúdo no CSS. A ideia foi criar um menu com um logo e três itens, além de aplicar efeito grayscale no hover das imagens e adicionar um footer. O flexbox permitiu criar um layout complexo e flexível ao mesmo tempo.

RESPONSIVIDADE

#HTML #CSS

Ver Repositório

Neste projeto trabalhei com o conceito de Responsividade. Ela permite que o seu projeto se ajuste automaticamente a diversos dispositivos presentes no mercado. Tendo a natureza como inspiração, selecionei algumas imagens e inclui em cada 'box' um campo para a legenda das fotos. O layout responsivo permite que o seu site seja acessível.

HTML - CSS - RESPONSIVIDADE

#HTML #CSS

Ver Repositório

Aprofundando ainda mais o conceito da Responsividade e o mobile first, este projeto foi uma homenagem à Seleção Feminina de Futebol. Foram adicionados: trailer, artigos, rodapé, fontes do Google Fonts, um botão para substituir o vídeo em telas menores e efeito de zoom nos ícones com efeitos suaves de entrada e saída.

INTRODUÇÃO À JAVASCRIPT

#HTML #CSS #JAVASCRIPT

Ver Repositório

Trabalhando com a linguagem de programação de alto nível, JavaScript, foi possível desenvolver um programa que solicita ao usuário um número inteiro que atenda a critérios específicos. Neste projeto, o JavaScript permitiu um comportamento dinâmico e com interatividade na página, para aprendizagem infantil.

JAVASCRIPT - DOM

#HTML #CSS #JAVASCRIPT

Ver Repositório

Conhecendo a interface de programação DOM, desenvolvi uma calculadora que realiza soma, subtração, divisão e multiplicação e exibe o resultado correto para o usuário. Ao usar o DOM, foi possível criar uma página web interativa, tornando-a mais dinâmica. Ele permite que o JavaScript acesse, modifique e manipule os elementos do documento.

MENU HAMBÚRGUER

#HTML #CSS #JAVASCRIPT

Ver Repositório

O menu hambúrguer permite que o usuário tenha uma boa experiência na navegação ao ocultar os links de navegação na página principal. Criei um site com um menu hambúrguer que se adapta às telas menores, ficando oculto quando exibido em telas maiores de forma responsiva.

BOOTSTRAP

#HTML #CSS #JAVASCRIPT

Ver Repositório

Aplicando o framework Bootstrap, adicionei uma galeria com o efeito carrossel. Essa galeria é acionada clicando no botão da página principal, sendo direcionada para uma nova guia que se abre com o carrossel de imagens. O Bootstrap fornece a criação de sites e aplicações responsivas de forma rápida e simples.
